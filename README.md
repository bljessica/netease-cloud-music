# netease-cloud-music

> 此项目为网易云音乐app前端界面，后端使用[网易云音乐NodeJS版API](https://binaryify.github.io/NeteaseCloudMusicApi/#/)

## 技术
+ vue+vuex+vue-router
+ 使用ElementUI的部分组件
+ 使用better-scroll滚动插件

## 运行
```
#切换到项目目录
cd netease-cloud-music

# 安装依赖
npm install

# 本地运行
npm run dev

# 生产打包
npm run build

```
